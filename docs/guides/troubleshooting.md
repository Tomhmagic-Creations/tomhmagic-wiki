# Troubleshooting

## Import Failed

- Confirm the file extension is `.mcpack` or `.mcaddon`.
- Download the file again in case the first download was incomplete.
- Make sure the Add-On supports your Minecraft Bedrock version.

## Add-On Does Not Work

- Confirm both behavior and resource packs are active if the Add-On uses both.
- Move the Add-On above conflicting packs in the world pack order.
- Enable the experiments listed on the Add-On page.
- Check whether the Add-On is compatible with Realms or servers.

## Missing Textures

Missing textures usually mean the resource pack is not active or the wrong version is installed.

## Missing Behaviors

If items, blocks, entities, or scripts do not work, confirm the behavior pack is active and required experiments are enabled.

## World Will Not Load

Restore a backup and check the Add-On page for migration notes or known issues.
