# Page Standards

Use consistent page structure so players can find the same information on every Add-On.

## Required Sections

- Status
- Download
- Installation
- Features
- Compatibility
- Known Issues
- Changelog

## Compatibility Table

<div class="compat-table">

## Writing Style

- Use direct player-facing instructions.
- Prefer short steps over long paragraphs.
- Note when a feature requires experiments.
- Keep version numbers exact.
